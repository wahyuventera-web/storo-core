📸 Cara Screenshot Banners untuk WhatsApp:

1. Buka file HTML ini di browser Chrome/Firefox
2. Klik kanan pada setiap banner → "Inspect Element"
3. Klik kanan pada element banner → "Capture node screenshot"
4. Atau gunakan Windows Snipping Tool (Windows + Shift + S) untuk screenshot manual
5. Save sebagai JPG dengan nama yang sesuai

Alternative: Install browser extension "GoFullPage" atau "Full Page Screen Capture" untuk screenshot otomatis
