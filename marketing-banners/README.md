# Marketing Banners untuk Storo.id

Banner marketing ini dirancang untuk digunakan di WhatsApp dan media sosial lainnya.

## File Banner

1. **1-how-storo-works.html** - Menjelaskan 3 langkah mudah menggunakan Storo (Landscape 1080px)
2. **2-pricing-catalog.html** - Menampilkan 3 paket harga (Basic, Pro, Enterprise) (Landscape 1080px)
3. **3-why-storo.html** - Menjelaskan keunggulan Storo vs marketplace (Landscape 1080px)
4. **4-whatsapp-cta.html** - Call-to-action untuk hubungi via WhatsApp (Square 1080x1080px)

## Cara Menggunakan

1. Buka file HTML di browser
2. Screenshot atau export sebagai gambar
3. Bagikan di WhatsApp atau platform lainnya

## Tips Screenshot

1. **Chrome/Firefox**: Tekan F12 → Toggle device toolbar → Set width 1080px → Screenshot
2. **Safari**: Develop → Enter Responsive Design Mode → Set width 1080px → Screenshot
3. Atau gunakan extension seperti "Full Page Screen Capture"

## Branding

Semua banner menggunakan:
- Logo resmi dari `/public/logo.png`
- Warna brand: Blue (#4169df) & Orange (#f3973b)
- Font: Inter
- Design system sesuai dengan website utama
