## Fitur & Layanan Utama

### 1. Import Produk dari Shopee
- Import langsung dari file Excel Shopee Seller Center (6 jenis file: Basic Info, Sales Info, Shipping Info, Media Info, DTS Info, Republish)
- Mendukung hingga 10.000 produk per import
- Gambar produk otomatis didownload ke CDN Storo
- Tidak perlu input manual satu per satu

### 2. Dashboard Admin Lengkap
- Manajemen produk (tambah, edit, hapus, varian, gambar, status)
- Manajemen pesanan (tracking status, detail pelanggan, cetak invoice)
- Manajemen kategori (tree-view, hierarki)
- Manajemen pelanggan (data kontak, riwayat pesanan)
- Statistik overview (total produk, pesanan, pendapatan, pelanggan)

### 3. Payment Gateway Terintegrasi
- **Midtrans:** E-wallet (GoPay, OVO, DANA, LinkAja), Virtual Account (BCA, BRI, BNI, Mandiri), retail (Alfamart, Indomaret), kartu kredit/debit
- **Xendit:** Alternatif payment provider
- Mode sandbox & production tersedia

### 4. Ongkir Otomatis (RajaOngkir)
- Kalkulasi ongkir real-time
- Kurir: JNE (REG, YES), J&T (EZ, Express), SiCepat (REG, BEST)
- Cakupan seluruh provinsi & kota di Indonesia
- Estimasi waktu pengiriman otomatis

### 5. Tema Profesional
- 6 pilihan tema: Starter, Modern, Fashion, Electronics, Grocery, Marketplace
- Kustomisasi branding (logo, banner, warna)
- Responsive (mobile-friendly)
- Dark/light mode

### 6. Toko Online (Frontend)
- Homepage dengan produk unggulan
- Halaman katalog & pencarian produk
- Halaman detail produk (galeri zoom, varian, stok)
- Keranjang belanja & checkout multi-step
- Halaman konfirmasi pesanan

### 7. Fitur AI (Segera Hadir)
- AI Product Description Enhancer
- AI SEO Generator (meta title, description, og:tags)
- AI Store Chatbot (chatbot toko berbasis katalog produk)
- AI Image Optimization

### 8. Keamanan & Autentikasi
- Login aman via Clerk
- Row Level Security (RLS) di database
- Enkripsi data sensitif (API keys, kredensial pembayaran)

### 9. Custom Domain
- Tersedia di paket Business & Enterprise
- Default: namatoko.storo.id

## Cara Kerja (Alur Penggunaan)

### Langkah 1: Daftar Gratis
Buat akun di Storo Engine — gratis, tanpa kartu kredit.

### Langkah 2: Import Produk dari Shopee
1. Login ke Shopee Seller Center (seller.shopee.co.id)
2. Buka "Produk Saya" → "Mass Update"
3. Export/Generate 6 file Excel
4. Upload ke halaman Import di Storo Engine
5. Produk otomatis tersinkronisasi

### Langkah 3: Kustomisasi Toko
Pilih tema, upload logo, atur nama toko & branding.

### Langkah 4: Konfigurasi Pembayaran & Pengiriman
Hubungkan Midtrans/Xendit dan atur kota asal pengiriman.

### Langkah 5: Toko Live!
Webstore siap menerima pelanggan. Kelola semua dari dashboard admin.

## FAQ (Pertanyaan yang Sering Diajukan)

### Umum
**Q: Apa itu Storo Engine?**
A: Storo Engine adalah platform yang membantu seller Shopee membangun webstore profesional sendiri. Produk dari Shopee bisa langsung diimport tanpa perlu input manual.

**Q: Apakah saya harus bisa coding?**
A: Tidak. Storo Engine dirancang agar siapa pun bisa membuat webstore tanpa kemampuan teknis.

**Q: Apakah saya harus menutup toko Shopee saya?**
A: Tidak. Anda bisa tetap berjualan di Shopee sambil punya webstore sendiri. Storo membantu Anda diversifikasi channel penjualan.

### Import & Produk
**Q: Bagaimana cara import produk dari Shopee?**
A: Login ke Shopee Seller Center → Produk Saya → Mass Update → Export 6 file Excel → Upload ke Storo. Produk akan otomatis masuk beserta gambar.

**Q: Berapa banyak produk yang bisa diimport?**
A: Starter: 50 produk, Business: 500 produk, Enterprise: unlimited. Per sesi import mendukung hingga 10.000 produk.

**Q: Apakah bisa import ulang jika ada update produk?**
A: Bisa. Upload file Excel terbaru dan sistem akan mendeteksi perubahan, menambah produk baru, dan mengupdate yang sudah ada.

### Harga & Pembayaran
**Q: Apakah ada biaya komisi per transaksi?**
A: Tidak. Storo tidak memungut komisi dari penjualan Anda, berbeda dengan marketplace.

**Q: Metode pembayaran apa saja yang tersedia?**
A: E-wallet (GoPay, OVO, DANA, LinkAja), Virtual Account (BCA, BRI, BNI, Mandiri), gerai retail (Alfamart, Indomaret), dan kartu kredit/debit.

**Q: Bisa pakai Midtrans atau Xendit?**
A: Bisa keduanya. Anda tinggal memasukkan API key di halaman Settings.

### Pengiriman
**Q: Kurir apa saja yang didukung?**
A: JNE (REG, YES), J&T (EZ, Express), dan SiCepat (REG, BEST). Ongkir dihitung otomatis saat checkout.

**Q: Apakah ongkir dihitung otomatis?**
A: Ya, menggunakan RajaOngkir. Pelanggan tinggal memilih kota tujuan dan kurir, ongkir langsung tampil beserta estimasi waktu.

### Teknis
**Q: Apakah toko saya bisa pakai domain sendiri?**
A: Bisa, tersedia di paket Business dan Enterprise. Default domain: namatoko.storo.id.

**Q: Apakah ada fitur AI?**
A: Fitur AI sedang dalam pengembangan, termasuk AI deskripsi produk, AI SEO, AI chatbot toko, dan AI optimasi gambar.

**Q: Bagaimana keamanan data saya?**
A: Storo menggunakan enkripsi untuk data sensitif, Row Level Security di database, dan autentikasi aman via Clerk.
